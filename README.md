# Lunar-Stray-Clans-Survivor.io

Competitive analysis data for 13 top clans in Survivor.io

## Files

- **lunar_clan_intelligence_TIMESTAMP.csv** - Clan rankings and stats
- **lunar_member_intelligence_TIMESTAMP.csv** - Individual member data

## What's Inside

### Clan Data
- Global rankings
- Guild IDs and names  
- Attack power and relic cores
- Member counts

### Member Data
- Player names and clans
- Attack rankings
- Relic core rankings
- Individual stats

## Usage

Download the CSV files and open in Excel, Google Sheets, or any spreadsheet program for analysis.

## Data Collection

Collected from Garry Tools Lunar platform using automated scraping.
